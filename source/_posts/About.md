**头文件**

在2021.7以前，仅适用万能头，现在使用`iostream`

**namespace std**

习惯于：
```cpp
using namespace std;
cout << "???" << endl;
```
不同于：
```cpp
using std::endl;
cout<<"I AK IOI"<<endl;
```
**输入输出**

一般都不用快读，除非特殊情况直接套板子，看情况，最常用的是
```cpp
int n, a[101];
cin >> n;
for (int i = 1; i <= n; i++) {
	cin >> a[i];
}
```
不用`scanf`是因为编辑器不支持

快读常用板子：
```cpp
int init(){
    char c = getchar();
    int x = 0, f = 1;
    for (; c < '0' || c > '9'; c = getchar())
        if (c == '-') f = -1;
    for (; c >= '0' && c <= '9'; c = getchar())
        x = (x << 1) + (x << 3) + (c ^ 48);
    return x * f;
}
```
**堆、栈、队列、单调队列**

从不手写
```cpp
queue<int>qwq;
```
**括号**

习惯小括号后先一个 `{`，一堆写完之后 `}`

哪怕只有一行内容也要坚持使用大括号
```cpp
int main() {
	for (int i = 1; i <= n; i++) {
		cin >> a[i];
	}
}
```
**宏定义**

别问了，根本不用的

**长代码里的 `int` 和 `long long`**
```cpp
#define ll long long 
#define itn int 
```
**变量名称**

一般就是 `n` 和 `m` 什么的，~~喜欢忽悠自己~~，结构体一般都是`qwq` 和 `qaq`

**空格**

都是按照VS里面的来的，个人还是很喜欢的。

其他就没啥了。
